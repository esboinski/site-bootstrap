Calculadora de Juros

Calculadora de Juros

Uma calculadora intuitiva e eficiente para realizar cálculos de juros simples de forma rápida e precisa. Ideal para estudantes, investidores e qualquer pessoa que precise calcular rendimentos ou valores futuros com base em um capital inicial, como investimentos, empréstimos ou até mesmo antecipação de cheques. Basta inserir o valor inicial, a taxa de juros e o período para obter o resultado de forma instantânea.

E-commerce

Um e-commerce fictício desenvolvido para fins educacionais, simulando uma loja online completa. O projeto inclui funcionalidades como cadastro de produtos, carrinho de compras, sistema de login e integração com banco de dados. Criado com tecnologias modernas como PHP, MySQL, Bootstrap e JavaScript, esse projeto é ideal para quem deseja aprender sobre o desenvolvimento de plataformas de comércio eletrônico e a estruturação de um backend funcional.
